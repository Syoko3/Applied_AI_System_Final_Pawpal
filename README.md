### Setup Instructions

Running the app:
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python -m streamlit run app.py
```

Testing the app:
```bash
python -m pytest
```
